# منصة لغوي | Lughawi Platform 📜

منصة ترجمة سياقية متخصصة (LSP) مبنية لدمج علم اللغويات التطبيقية مع تقنيات معالجة اللغات الطبيعية (NLP) والذكاء الاصطناعي، لتوفير تراجم فائقة الدقة في خمسة مجالات سيادية تفشل فيها أنظمة الترجمة التقليدية.

---

## 🚀 المميزات الرئيسية للمنصة

* **🕌 المحرك الديني المقيد:** معجم إسلامي وفقهي يحتوي على أكثر من 80 مصطلحاً موثقاً من مراجع أكاديمية، مع فرض ترجمات سياقية دقيقة (مثل ترجمة التقوى إلى God-consciousness) وشرح الفروق الدلالية والتحذيرات الفقهية.
* **💻 المحرك التقني:** معجم مخصص لعلوم الحاسب، العتاد، الذكاء الاصطناعي التوليدي، وهندسة الأوامر.
* **🩺 المحرك الطبي والقانوني:** واجهات مهيأة للمصطلحات التخصصية وصياغة العقود والترميز الطبي.
* **✨ محرك التنبؤ الذكي:** نظام تنبؤ ذكي وخفيف (Predictive Text) لمساعدة المستخدم في العبارات اليومية الشائعة بالقسم العام.
* **🎨 تجربة مستخدم كلاسيكية فخمة:** تصميم راقٍ بعيد عن أسلوب الشات التقليدي للـ AI، يركز على راحة العين والأداء الأكاديمي السريع.

---

## 🛠️ البنية التحتية والتقنيات (Tech Stack)

* **Frontend:** React, TypeScript, Vite
* **Data Validation:** Zod Schema
* **AI Core:** Google Gemini (Optimized Prompts)
* **Deployment:** Fully optimized for zero-cost client-side deployment (Vercel).
